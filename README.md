A Spigot plugin that gives players limitless and organized item storage.

## Permissions
| Permission  | Description |
| ------------- | ------------- |
| infinitychest.access | Grants permission to own an infinity chest |
| infinitychest.sell  | Grants access to sell items from an infinity chest |
| infinitychest.give  | Grants access to give another player items from your infinity chest |
| infinitychest.autocollect  | Grants access to toggle whether items should automatically be stored to your infinity chest |
| infinitychest.slots.``[amount]``  | Sets the max amount of slots a player can have |
